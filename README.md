# PDF Catalog Parser
Скрипт предназначен для извлечения структурированных данных из PDF-каталога режущих пластин [Deskar_compressed.pdf](input%2FDeskar_compressed.pdf), в котором таблицы представлены в виде изображений, прямое копирование данных из такого PDF невозможно — используется распознавание текста с изображений и последующая нормализация данных.

## 🔍 Входные данные

PDF-каталог со страницами, содержащими:
- Модель и значения атрибутов — строки данных
- Атрибуты — заголовки столбцов таблицы
- Сплав — марка материала пластины

Особенность:
Если у одной модели указано несколько сплавов (например, одновременно M и K), в выходном файле создаются отдельные строки для каждого сплава.


## 📊 Выходной формат
Скрипт генерирует Excel-файл со следующей структурой (см. [result.xlsx](output%2Fresult.xlsx)):

|model|	L	|I_C	|S	|d|	r	|P	|M	|K|
|:---|:---|:---|:---|:---|:---|:---|:---|:---|
|CNMG120404-HQ|	12.9|	12.7|	4.76|	5.16|	0.4|	LF9008|		
|CNMG120408-HQ|	12.9|	12.7|	4.76|	5.16|	0.8|	LF9018		
|CNMG120404-TF|	12.9	|12.7|	4.76|	5.16|	0.8	|LF9018		
|...|	...|	...|	...|	...|	...|	...|	...|	...

Описание колонок:

model — модель пластины (например, CNMG120404-HQ)

L — длина

I_C — диаметр вписанной окружности

S — толщина

d — диаметр отверстия

r — радиус при вершине

P, M, K — группы обрабатываемых материалов (сплавы)


## ⚙️Логика обработки

1. PDF конвертируется в изображения.

2. Vision-модель извлекает:
- атрибуты
- модели
- параметры
- соответствие сплавов

3. Парсер нормализует данные:
- выравнивает колонки
- проверяет обязательные поля
- дублирует строки при множественных сплавах

4. Генерируется итоговый XLSX.


## 🚀 Установка и запуск

Клонируй репозиторий:
```bash
git clone https://github.com/MaminAmetist/pdf_parser.git
cd pdf_parser
```

Установи зависимости:
```bash
pip install -r requirements.txt
```

Создай файл [.env](.env) в корне проекта:

```python
AI_API_KEY =...
MODEL_NAME=...
```

Запусти скрипт:
```bash
python main.py
```

Результат: скрипт создаст result.xlsx в директории [output](output)


## ⚠️ Примечания
